Secure Shell

Description

The Secure Shell Language was created by the Open BSD foundation to allows for highly secure communication using complex routing techniques and cryptographic keys to enable remote access of host computers.

Terminal Required

Terminal is a text command interface available on UNIX like systems. It enables written commands to effect every function of the computer. Generating and accessing SSH key data requires use of the terminal. If this is the first time you are reading about the terminal I suggest that you read or watch a short tutorial on using your specific terminal before attempting SSH key generation.

SSH is needed

SSH enables trusted communication to devices on the internet. It creates a trusted key-pair that acts as an identifyable fingerprint for trusted exchange. This is especially important with GitHub where many users are accessing a remote server to contribute changes.
