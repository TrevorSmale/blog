Blender


Description

Blender is an Open Source software tool for creating 3D models and animations. Originally created by Ton Roosendaal as a fledgling modelling program, it has grown into a massively popular full featured suite with a physical headquarter building in Amsterdam. Thanks to the open nature of both the software and passionate community surrounding it, Blender has grown into such a capable toolset that it now stands toe to toe with major commercial software. Furthermore, the software is now being implemented into customized professional pipelines for film and animation. Outside of entertainment the software is now used in scientific research projects.



I use blender often

I had used the software for quick tasks during University Projects and Game Jams with only surface knowledge. After much preliminary research into Design software and speaking with others in my industry, came to the realization that much of the design work I had done in the past could be replicated in Blender. Moreover, I had realized that I would be able to integrate Blender with Rhino 3D, a really good NURBS design tool. Several years ago I took it upon myself to learn Blender to a level of depth where I would be comfortable with it enough to handle production work. Several years later, and I am glad with my time investment.

I have also come to realize that Blender with Plug-ins is capable of far more than I had originally anticaped and this is a fantastic bonus.

Development

The pace of development is feverish, there are so many inbuilt features, plugins and mods, that it seems like less of a singular program and more like an operating system capable of photo realistic everything. The efforts of the massive community have culminated into version 2.8, a very major release. In terms of overall user experience and functionality Version 2.8 represents a major milestone. 2.8 is currently in Beta and will be release near q4 2019. Never has there been a better time to get into using the software.

Contributing

If you are not familiar with Open Source software, there is a mantra amongst fervent enthusiasts. If you are using a free, open source software, then you must contribute in some way to the cause.

Financial Contributions

The Blender Foundation accepts donations in the form of membership

Development Contributions

Blender accepts edits to its software via pull request on GIT

Educational Contributions

Many advanced Blender users create free educational content for aspiring users

Editorial Contributions
